# dats-defense
Big Data Small Memory

## Requirements

- Python 3.12

## Setup

.env file
```bash
# .env
TELEGRAM_BOT_TOKEN=
TELEGRAM_CHAT_ID=
TELEGRAM_TOPIC_ID=
```

```bash
poetry install
```

## Run

```bash
poetry run python src/main.py
```

```bash
poetry run python src/main.py --command-test
```

```bash
# /mockgame
poetry run uvicorn app:app
```
